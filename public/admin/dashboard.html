Note:

This page can be used to monitor completion status of hoosiers attending an event and offers a way for officials to track the quality of Hoosier's assistance.


This page will also provide statistics about the event. Including:

  • Time of arrival (noted by when a hoosier starts a survey)
  • Time to complete survey.
  • Time to recieve asistance from each agency (revealing what stations need the most assistance)
  • The time it took to recieve all requested assistance.

Currently this portal gathers no personal information about a user and all hoosiers are identified by anonymous IDs to protect privacy.
